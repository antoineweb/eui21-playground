<button euiButton> — source of truth

Selector: button[euiButton], a[euiButton] · Module: @eui/components/eui-button · Export: EUI_BUTTON

Color variants

One BaseStatesDirective attribute per variant. Combine at most one.

Outline (ghost) style

Add euiOutline to any color variant.

Sizes

XS · S · M (default) · L. Use one per button.

Shape & layout modifiers

euiBlockButton (full width)

Special roles

States

Content alignment within button

Combine with euiBlockButton to see euiStart / euiEnd in action.

Element types

The directive matches both button and a.

<a euiButton href="…">

(buttonClick) output

Emits the EuiButtonComponent instance. Last click → {{ lastClick() }}
